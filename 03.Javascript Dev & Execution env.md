# 자바스크립트 실행 환경

```
웹 자바스크립트 와 NodeJS의 차이
```

-   브라우저 자바스크립트 엔진과 NodeJS
-   브라우저 : HTML CSS JS 실행, 웹 렌더링
    -   파일 시스템 제공 안함. (클라이언트 사이드 웹 API 제공)
-   NodeJS : 브라우저 외부에서 실행환경 제공
    -   DOM API 제공 안함. (NodeJS 고유 API 제공)
-   ECMAScript 공통 제공
